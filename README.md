# MyAgri

Portail citoyen multilingue d'information sur l'agriculture en Wallonie.

## Langues

- Français : `?lang=fr`
- Anglais : `?lang=en`
- Allemand : `?lang=ge`
- Néerlandais : `?lang=nl`

Les contenus éditoriaux sont séparés dans `includes/translations/`.

## Validation

- `php tests\smoke.php`
- `php scripts\build_sitemap.php`
- `php -l` sur les fichiers PHP modifiés
